API Basejump: Image Search Abstraction Layer

User stories:
    1) I can get the image URLs, snippets and page urls for a set of images relating to a given search string.
    2) I can paginate through the responses by adding a ?offset=2 parameter to the URL.
    2) I can get a list of the most recently submitted search strings.

Example usage:

Image search

https://search4image.herokuapp.com/api/imagesearch/lolcats%20funny?offset=8

Latest search strings

https://search4image.herokuapp.com/api/latest/imagesearch/
